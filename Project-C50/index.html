CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata).

  • O CSS é a linguagem que usamos para estilizar um documento HTML.
  • O CSS descreve como os elementos HTML devem ser exibidos.
  • O CSS economiza muito trabalho. Ele pode controlar o layout de várias páginas da web de uma só vez.
